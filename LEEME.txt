1. Sube esta carpeta a un repo GitHub (puede ser solo render-deploy/ como raiz del repo).
2. En Render: New > Blueprint y selecciona render.yaml
   O crea dos servicios (API Node en api/, Static Site en web/).
3. IMPORTANTE — proxy same-origin (/api):
   En el servicio web (drewrest-demo-web) → Redirects/Rewrites, añade AL INICIO:
     Source: /api/*
     Destination: https://drewrest-demo-api.onrender.com/*
     Action: Rewrite
   (Si usas Blueprint: Sync del Blueprint tras el push para aplicar routes del yaml.)
4. CORS_ORIGINS del API debe incluir la URL publica de la web.
5. Actualizar: npm run Render:Publicar-Demo (o Empaquetar + push).
