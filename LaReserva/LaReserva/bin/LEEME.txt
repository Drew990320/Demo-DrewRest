Herramientas de La Reserva (uso ocasional o soporte)
====================================================

Uso diario (en la carpeta padre LaReserva\):
  abrir-la-reserva.vbs       Abrir la app (sin ventanas CMD)
  crear-accesos-directos.vbs Acceso directo en el escritorio

En esta carpeta bin\:
  inicio.bat                 Arrancar con ventanas CMD (ver errores)
  detener.bat                Detener servidor (ventana CMD)
  detener.vbs                Detener sin ventana CMD
  mostrar-id-maquina.bat     ID de este PC para activar la licencia
  evitar-suspension.bat      Evitar que Windows suspenda el PC
  abrir-firewall.bat         Abrir puertos 3000 y 8080 (admin)
  reparar-api.bat            Reinstalar dependencias del API
  ejecutar-baseline-prisma.bat   Solo si migrate falla con P3005
  ejecutar-fix-cobro-parcial.bat Migracion puntual (soporte)

Celulares en la red: admin - Conexion movil (QR y URL en la app).

Scripts internos (PowerShell): carpeta ..\scripts\
